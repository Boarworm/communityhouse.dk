## 🛠️ Theme Configuration

1. Generate favicons at [realfavicongenerator.net](https://realfavicongenerator.net) and upload them to:
   `themes/main/assets/favicons`

2. Paste the HTML code generated by realfavicongenerator.net into:
   **Site > Config > General > Favicons**

## 🛠️ Development

Open a terminal in the theme folder and run:

```bash
  npm install
```

Compile the assets using one of the following:

```bash
  npm run dev     # watch mode with HMR (Vite dev server on port 5173)
  npm run build   # production build
```

Inside Docker:

```bash
  docker compose exec app sh -c "cd themes/main && npm run dev"
  docker compose exec app sh -c "cd themes/main && npm run build"
```

> **Note:** When running `npm run dev`, set `VITE_DEV_ENVS=local` in `.env`
> so the plugin connects to the Vite dev server instead of serving from
> the compiled manifest. When you're done, run `npm run build` and set
> `VITE_DEV_ENVS=` (empty) to switch back to serving built assets.
